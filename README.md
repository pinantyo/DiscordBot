# DiscordBot - Question Answering System

## About
Discord Bot for helping retrieving information related to the company.
Methods implemented:
- Sentence Similarity
- Open-domain Retrieval Question Answering System
- GPT Integration

## Stack

- Discord.py
- Sklearn
- Flask
